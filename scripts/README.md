# Scripts

Utility scripts for development, testing, and deployment.

## Development Scripts

- `setup_dev.sh` - Set up development environment
- `run_tests.sh` - Run full test suite with coverage
- `format_code.sh` - Format code with Black and isort
- `check_code.sh` - Run linters (flake8, mypy)

## Deployment Scripts

- `build_package.sh` - Build distribution packages
- `publish_pypi.sh` - Publish to PyPI (maintainers only)
- `create_release.sh` - Create GitHub release

## Benchmarking Scripts

- `benchmark_accuracy.py` - Test prediction accuracy on benchmarks
- `benchmark_performance.py` - Measure execution time
- `validate_model.py` - Validate ML model on test set

Scripts will be added during development and release preparation.
